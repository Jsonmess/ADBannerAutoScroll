# ADBannerAutoScroll
用于展示Banner广告，实现自动轮播，无限循环滚动功能.

新增:

1.支持本地和在线banenr图片的展示（在线url 使用SDWebimage 下载);

2.添加自动展示banner 定时的控制;

3.添加自动展示banner 滚动方向展示 （左/右）;
